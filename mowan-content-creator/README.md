# ✍️ 墨玩内容创作（mowan-content-creator）

> 一个想法，一千种写法——帮你从热帖中挖素材、找角度、攒灵感

---

## 这是什么

墨玩内容创作是一个 Claude Code Skill，帮助创作者系统化地分析热帖、挖掘素材、生成写作切入点，把零散的灵感变成可执行的写作方案。

核心理念：**一个想法一千种写法**。同一个核心观点，用不同钩子重新包装，适配不同平台和受众。

---

## 功能

| 命令 | 说明 |
|------|------|
| `分析 <链接或文本>` | 全面分析热帖：核心想法、句子结构、心理策略、复刻路径 |
| `素材 <链接或文本>` | 从内容中提取高密度想法、独特视角、情绪触发点 |
| `切入点 <主题>` | 为同一主题生成多个不同角度的写作方案 |
| `归档 <内容>` | 格式化存入想法博物馆，方便日后检索 |

## 分析维度

### 热帖分析

- 想法密度：一个帖子传递了多少有价值的想法
- 核心观点：作者真正想表达的一个核心信念
- 交叉点：作者如何组合不同领域的知识
- 心理策略：为什么人们会与之互动
- 复刻路径：如何一步步复刻这种风格

### 素材挖掘

- 高密度想法：一句话就能让人顿悟
- 独特视角：怎么把普通话题讲出新意
- 情绪触发点：哪些内容让你有强烈反应

### 写作切入点

- 同一核心观点，不同钩子包装
- 考虑不同平台的内容形式
- 结合你自己的经历和视角

| 写作切入点 | `内容创作/写作切入点/{{日期}}-{{主题}}.md` |

- 所有分析在本地完成，不上传任何数据到外部服务器
- 归档内容仅保存在你指定的本地目录
- 不收集任何用户数据
