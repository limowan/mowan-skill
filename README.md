## 🚀 我的 Skills 列表

### 1. mowan-mc-type（墨成AI排版）

**功能**：将 Markdown 文章转换为微信公众号精美排版的 HTML

**特点**：
- 6种风格预设（极简理性、温暖故事感、商务专业、编辑长文风、视觉杂志大片、新媒体高亮风）
- 自动生成滑动卡片导航、引导关注卡片
- 一键复制到公众号编辑器
- 输出内联样式 HTML，兼容性好

**使用场景**：公众号文章排版

### 2. mowan-sucai-sort（墨玩素材整理）

**功能**：自动整理素材库，建立分类索引，方便后续AI检索

**特点**：
- 智能探测素材库位置，首次使用引导建立分类结构
- 支持微信读书笔记、公众号文章、小红书笔记、个人复盘等多种素材类型
- 增量更新：基于文件修改日期，只处理新增和修改的文件
- 双文件夹结构：「内容文件夹 + 分类索引文件夹」安全可控
- 自动检测文件夹改名、移动等变动

**使用场景**：管理微信读书笔记、公众号文章、小红书笔记、个人复盘等

### 5. mowan-CTI（墨玩CTI人格侧写）

**特点**：
- 12 种独特聊天人格分类（改改改改、凌晨三点兽、散装弹珠、随便侠等）

- 生成毒舌风格的人格结果页
