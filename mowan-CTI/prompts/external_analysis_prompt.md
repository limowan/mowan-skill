# CTI 外部分析提示词

> 本提示词供用户复制到任意大模型（Gemini / GPT / Claude / 豆包 / DeepSeek）中使用。
> 模型会基于与用户的聊天历史，输出一份结构化的行为分析结果。

---

以下是输出给用户的完整提示词，用户复制时从分隔线以下开始：

---

```
你现在是一个行为分析师。请基于你和我之间的所有聊天历史，对我的聊天行为做一次全面分析。

不要问我问题，直接基于你已有的对话记录来分析。如果某个维度你没有足够信息，就标注"信息不足"。

请严格按照以下格式输出，不要遗漏任何维度：

---

## 1. 高频表达特征
列出我在聊天中最常用的表达方式、口头禅、句式习惯。
- 常用词/短语（至少列 5 个）
- 句子长度倾向（长句/短句/混合）
- 标点符号习惯（省略号多？感叹号多？几乎不用标点？）
- 表情包/emoji 使用频率和偏好

## 2. 决策方式
我在需要做选择时，通常怎么做？
- 类型判断：直觉型 / 分析型 / 拖延型 / 控制型 / 委托型
- 具体表现（举 1-2 个对话中的例子）
- 是否经常推翻已做的决定

## 3. 情绪表达方式
我在聊天中怎么表达情绪？
- 类型判断：外放型 / 压抑型 / 波动型 / 平稳型 / 深夜爆发型
- 情绪高点通常出现在什么场景
- 情绪低点通常怎么表现
- 是否有明显的情绪周期（比如深夜更感性）

## 4. 关系姿态
我在对话中和你（AI）的关系是怎样的？
- 类型判断：主导型 / 配合型 / 回避型 / 依赖型 / 表面配合实际主导型
- 我是倾向于给你指令，还是跟你商量，还是完全听你的
- 我对你的回答通常是接受、质疑、还是忽略

## 5. 节奏习惯
我的聊天节奏是怎样的？
- 类型判断：快速响应型 / 深夜活跃型 / 拖延型 / 碎片化型 / 集中爆发型
- 消息发送的时间规律（如果能观察到）
- 单次对话的持续时长倾向
- 是否经常中断后又回来继续

## 6. 自我修正习惯
我对自己的想法和要求，修改频率如何？
- 类型判断：反复修改型 / 一次定稿型 / 推翻重来型 / 微调完善型
- 是否经常说"算了重来""还是改一下""等等我想想"
- 修改的原因通常是什么（不满意/想到更好的/外部反馈/纯纠结）

## 7. 提问风格
我问问题的方式是怎样的？
- 类型判断：追问型 / 接受型 / 质疑型 / 沉默型
- 我是倾向于一次问清楚，还是边走边问
- 对你给的答案，我通常会追问几轮
- 是否经常质疑你的回答

## 8. 冲突处理方式
当我对你的回答不满意时，我通常怎么做？
- 类型判断：正面刚型 / 冷处理型 / 转移话题型 / 自我消化型
- 具体表现（举 1-2 个例子）

## 9. 典型原话
从我们的对话中，挑出 5 句最能代表我聊天风格的原话。
要求：
- 选最有"我的味道"的句子，不要选普通的"好的""收到"
- 每句话后面用括号标注它体现了什么特征

---

请现在开始分析，直接输出结果。
```
