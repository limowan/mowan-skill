# 索引模板

本文件定义各类素材的索引条目格式模板。

---

## 通用索引结构

每个分类索引文件应包含以下结构：

```markdown
# [分类名称]索引

## [子分类1]

- 文件名.md
  - 字段1：值
  - 字段2：值

## [子分类2]

- 文件名.md
  - 字段1：值
  - 字段2：值
```

## _config.md 模板

每个分类索引文件夹根目录下的配置文件：

```markdown
# 索引配置

根目录: [相对于素材库根目录的内容文件夹路径]

---

## 索引说明

- 本索引通过 AI 自动生成，用于快速检索素材
- 索引条目只包含文件名，完整路径 = 根目录 + 文件名
- 最后更新时间：[自动填充]
```

### 示例

```markdown
# 索引配置

根目录: 微信读书笔记和摘录（截止26-1-22）

---

## 索引说明

- 本索引通过 AI 自动生成，用于快速检索素材
- 索引条目只包含文件名，完整路径 = 根目录 + 文件名
- 最后更新时间：2025-04-16 15:30:00
```

## 新增条目追加格式

更新模式下，在分类文件末尾追加新条目：

```markdown

<!-- 本次更新：2025-04-16 -->

- [2025-04-16]新文章标题.md
  - 核心观点：新文章核心观点
  - 标签：#标签1 #标签2
```
