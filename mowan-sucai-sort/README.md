# 墨玩素材整理助手 (mowan-sucai-sort)

> 让AI帮你自动整理素材库，建立分类索引，从此告别「素材找不到」的困扰。

---

## 功能亮点

| 特性 | 说明 |
|------|------|
| 🔍 **智能探测** | 自动探测素材库位置，首次使用引导建立分类结构 |
| 🏷️ **自动分类** | 支持微信读书笔记、公众号文章、小红书笔记、个人复盘等多种素材类型 |
| ⚡ **增量更新** | 基于文件修改日期，只处理新增和修改的文件，高效快捷 |
| 🗂️ **双文件夹结构** | 「内容文件夹 + 分类索引文件夹」配对，索引与原始素材分离，安全可控 |
| 📝 **Markdown索引** | 生成标准Markdown格式的分类索引，任何编辑器都能打开 |
| 🔁 **变动检测** | 自动检测文件夹改名、移动等变动，保持索引同步 |

---

## 适用场景

- 📚 **微信读书笔记** → 按书籍、主题、阅读时间分类
- 📝 **公众号文章** → 按主题、发布时间、专栏分类
- 📸 **小红书笔记** → 按目的地、主题分类
- 🔄 **个人复盘** → 按时间（周/月/年）、项目分类
- 🌐 **网页摘录** → 按主题、来源分类

## 快速开始

### 安装

告诉你的 AI 助手：

> 帮我安装一下 https://github.com/limowan/mowan-skill/tree/main/mowan-sucai-sort 这个 Skill

支持 Claude Code、Cursor 或其他兼容的 AI 编程工具。

### 使用方法

安装后，在对话中输入：

```
整理素材
```

或

```
更新素材索引
```

### 首次使用流程

1. **探测素材库位置** → AI自动探测或询问你的素材库路径
2. **选择素材类型** → 选择需要整理的素材类型（可多选）
3. **确认分类维度** → 选择按什么维度分类（如按主题、按时间等）
4. **自动创建结构** → AI自动创建分类索引文件夹和配置文件
5. **生成索引** → 分析所有素材，生成分类索引
6. **查看报告** → 输出整理完成报告，包含变更统计和索引位置

### 后续使用

只需一句话，AI自动检测变动并增量更新：

```
更新素材索引
```

```
┌─────────────────────────────────────────────────────────┐
│  1. 探测素材库位置                                        │
│     └─ 自动探测常见路径 / 询问用户指定路径                    │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  2. 首次初始化（新用户）                                   │
│     └─ 选择素材类型 → 确认分类维度 → 创建目录结构             │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  3. 扫描目录结构                                          │
│     └─ 识别「内容文件夹 + 分类索引文件夹」配对                │

└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  4. 增量更新                                              │
│     └─ 对比文件修改日期，只处理新增/修改的文件               │
│     └─ 读取资源文件中的分类规则                            │
│     └─ 分析文件内容，提取核心观点                          │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  5. 生成索引                                              │
│     └─ 在分类索引文件夹中生成/更新 Markdown 索引文件         │
│     └─ 记录更新日志                                       │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│  6. 输出报告                                              │
│     └─ 变更统计 / 索引位置 / 使用说明                        │
└─────────────────────────────────────────────────────────┘
```

## 配置说明

### `_config.md` 配置文件

每个分类索引文件夹根目录下的 `_config.md` 文件定义了该索引对应的内容文件夹路径：

```markdown
# 索引配置

根目录: 微信读书笔记和摘录

---

## 索引说明

- 本索引通过 AI 自动生成，用于快速检索素材
- 索引条目只包含文件名，完整路径 = 根目录 + 文件名
- 最后更新时间：2025-04-16 15:30:00
```

**重要**：当内容文件夹改名时，**只需修改 `_config.md` 中的根目录**，索引条目无需改动。

## 更新日志

更新日志按素材类型分别存储：

```
素材自动更新记录/
├── 微信读书笔记素材更新记录/
│   └── 2025-04-16_15-30_更新日志.md
├── 公众号文章素材更新记录/
│   └── 2025-04-16_16-00_更新日志.md
└── ...
```

每篇日志包含：
- 更新时间
- 变更统计（新增/修改/删除数量）
- 新增文件详情
- 目录结构快照（用于下次对比发现变动）

---

## 注意事项

1. **不修改原始素材**：只创建/更新分类索引文件夹，不动原始素材文件
2. **索引与素材分离**：分类索引文件夹和原始素材文件夹是分开的，安全可控
3. **基于文件修改日期**：更新模式下，自动识别新增和修改的文件
4. **自适应目录结构**：每次使用前扫描目录，不硬编码路径
5. **路径变动检测**：自动检测文件夹改名、移动等变动
6. **增量更新**：只处理有变动的文件，高效快捷
