# 大道投研 · Dadao Investment Research

> 基于段永平投资哲学的 AI 决策框架

一套从段永平 50+ 一手来源中提炼的投资决策工具，帮你用"做对的事情"的思维审视每一个投资和商业决策。

---

## 这是什么

**大道投研**是一个 [Claude Agent Skill](https://docs.anthropic.com/en/docs/claude-code/skills)，可以在 Cursor / Claude Code 等支持 Agent Skill 的环境中使用。

它不是一个人物模拟器，而是一套**实用的投资决策检查清单**。给你一家公司或商业机会，按流程走一遍，帮你判断值不值得深入研究。

### 两种使用模式

| 模式 | 说明 | 触发方式 |
|------|------|---------|
| **决策流程** | 三关检验 → 红线清单 → 买卖判断，结构化分析 | 默认模式 |
| **对话模式** | 以段永平视角与你对话，用他的语气和思维方式回应 | 说"切换对话模式" |

---

## 核心功能

### 三关检验

1. **好生意（Right Business）** — 商业模式十年后还在不在？定价能力、抗变化能力、资本支出
2. **好人（Right People）** — 管理层的诚信正直够不够？企业文化、利益相关方关系
3. **好价格（Right Price）** — 你对这家公司的理解深度，够不够支撑安全边际？

> 任何一关不过，直接放弃，不要被价格诱惑。

### 红线清单（8条投资禁区）

触发任何一条，不碰：
- 说不清怎么赚钱 / 需要问别人能不能买 / 靠价格竞争 / 管理层有诚信污点
- 需要借钱投资 / 想赚快钱 / 靠补贴烧钱 / 因为别人在买

### 商业模式判断工具

5个核心问题，不只看股票，看任何商业机会都能用。

## 使用示例

### 决策流程模式

> **你**：帮我用大道投研的框架分析一下泡泡玛特
>
> **AI**：按三关检验分析——第一关（好生意）：毛利72%说明有定价能力，但LABUBU单IP占40%收入是集中度风险...

### 对话模式

> **你**：切换对话模式。某某公司在搞AI+新能源，很火，能买吗？
>
> **段永平**：你先告诉我，它怎么赚钱的？说不清楚就不要碰...
